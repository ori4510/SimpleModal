About

SimpleModal - плагін для створення і для роботи з модальними вікнами.

SimpleModal можна використовувати 2 шляхами:

  1. Метод createModal();
    Створити модальне вікно динамічно, передавши аргументи title і content. Все буде згенеровано JS і додано в DOM-дерево.
  2. Метод getModal();
    Статично. Додати десь в межах тега <body> html-шаблон модального вікна з вашим контентом. Приклад тут

SimpleModal має 3 типи:

Просте модальне вікно

Звичайне просте модальне вікно, контентом може бути що завгодно: текст, форма, таблиця, зображення і т.д.


Модальне вікно з підтвердженням

Має 2 кнопки 'Ok' та 'Cancel'. Можна використовувати якщо потрібна якась дія з підтвердженням, щось видалити, наприклад. Також можна додати title і/або content.

Підтвердження спрацює при натисканні клавіши Enter, а також клікнувши по кнопці 'Ok'. Після чого запуститься callback onConfirm, якщо передати в нього функцію.

Відміна спрацює лише клікнувши по кнопці 'Cancel'. Після чого запуститься callback onCancel, якщо передати в нього функцію. Якщо бажаєте також задіяти натискання клавіші Escape, а також клікнувши по масці чи по close-button для розпізнавання відміни використайте додатково метод onClose.


Інформаційне вікно

Модальне вікно з кнопкою 'Ok', яка просто закриває модальне вікно. Можна на неї повісити якусь свою подію.



Обов'язковий параметр лише один - modalId. Якщо використовуєте createModal() тоді modalId потрібно придумати. Дублі id призведуть до помилки. А якщо використовуєте getModal() тоді modalId повинен відповідати id яке в html-шаблоні.




Відкрити модальне вікно

Варіант 1. Визвати метод openModal() у змінної з екземпляром класу SimpleModal.


Крок 1. Створіть [createModal()] або отримайте [getModal()] модальне вікно і помістіть екземпляр у змінну.

copyconst myModal = Modal({
	modalId: 'myModalId',
	title: 'My title',
	content: 'My content',
}).createModal();

Крок 2. Викличте метод openModal(), щоб відкрити модальне вікно..

copydocument.querySelector('#button').addEventListener('click', () => {
	myModal.openModal();
});


Варіант 2. Через кнопки.


Step 1. Створити на сторінці кнопку чи кнопки, додати атрибут data-dom-modal, передавши в нього id необхідного модального вікна.

copy<button data-dom-modal="myModalId">Open modal window</button>

Step 2. Додати аргумент openWithBtn: true до виклику модального вікна.

copyconst myModal = Modal({
	...
	openWithBtn: true,
}).createModal();


Закрити модальне вікно

Щоб закрити модальне вікно, просто натисніть кнопку закриття, на маску або натисніть Escape. Але якщо модальне вікно містить, наприклад, форму, і після її відправки вам потрібно закрити модальне вікно, використовуйте метод closeModal().

copymyModal.closeModal();


Методи

onOpen

Запускається після відкриття модального вікна, якщо в нього передати функцію. Працює для всіх типів модальних вікон.


onClose

Запускається після закриття модального вікна, якщо в нього передати функцію. Працює для всіх типів модальних вікон.


onConfirm

Запускається після кліку на кнопку підтвердження у модальному вікні, якщо в нього передати функцію. Закриває модальне вікно. Працює тільки для type: 'confirm'.


onCancel

Запускається після кліку на кнопку відміни у модальному вікні, якщо в нього передати функцію. Закриває модальне вікно. Працює тільки для type: 'confirm'.


Нижче приклад роботи onOpen і onClose.
При відкритті модального вікна, яке містить форму, знаходимо перший інпут, не з типом "hidden" і додаємо йому фокус.
А при закритті модального вікна робимо ресет форми, якщо в інпутах раптом залишалась якась інфа при закритті вікна.

copyconst myModal = Modal({
	modalId: 'modal_example',
	onOpen: () => {
		document.querySelector('#modal_example form input:not([type="hidden"])').focus();
	},
	onClose: () => {
		document.querySelector('#modal_example form').reset();
	},
}).getModal();


Видалення

Щоб знищити екземпляр модального вікна і DOM-елемент використайте метод destroyModal()

copymyModal.destroyModal();

Simple modal

Example:

  • modalId - (string) ідентифікатор модального вікна.
  • title - (string) title модального вікна. Підтримує HTML-теги.
  • content - (string) content модального вікна. Підтримує HTML-теги.
  • type - (string) (simple | confirm | info) тип модального вікна.
  • openWithBtn - (boolean) дозволити відкривати модальне вікно з кнопки/кнопок вставлених на сторінці. Детальніше тут
  • wideModal - (boolean) додає клас 'dom-modal-wide', який розширяє максимальну ширину модального вікна до 1200px.
  • onOpen - (function) колбек-функція, яка запускається коли відкривається модальне вікно. Приклад тут
  • onClose - (function) колбек-функція, яка запускається коли закривається модальне вікно. Приклад тут

===== only for this type =====
  • modalConfirmBtnText - (string) текст для кнопки підтвердження. За замовчуванням 'Ok'
  • modalCancelBtnText - (string) текст для кнопки скасування. За замовчуванням 'Cancel'
  • onConfirm - (function) callback-функція, яка запускається після натискання кнопки підтвердження, а також при натисканні клавіші Enter
  • onCancel - (function) callback-функція, яка запускається після натискання кнопки відміни
copyconst myConfirmModal = Modal({
	modalId: 'myConfirmModalId',
	title: 'Confirm Modal',
	content: 'Content',
	type: 'confirm',
	modalConfirmBtnText: 'Yes',
	modalCancelBtnText: 'No',
	onConfirm: () => {
		alert('Confirmed');
	},
	onCancel: () => {
		alert('Canceled');
	},
}).createModal();

Якщо вам не потрібно або не хочеться створювати модальне вікно динамічно за допомогою JS, ви можете розмістити на сторінці HTML-шаблон модального вікна з вашими даними. Ви можете отримати його для подальшої роботи за допомогою методу getModal().


Крок 1. Вставте шаблон на свою сторінку. Не забудьте змінити id

copy<div class="dom-modal-mask" id="my_modal_id">
	<div class="dom-modal-window">
		<span class="dom-modal-exit dom-modal-close-with-click"><svg width="22" height="22" stroke-width="2" stroke="currentColor" stroke-linecap="round" viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg"><path d="M18 6 6 18M6 6l12 12"></path></svg></span>
		<div class="dom-modal-container">
			<h4 class="dom-modal-title">My_Modal_Title</h4>
			<div class="dom-modal-content">
				<!-- // your_content -->
			</div>

			<!-- for type confirm -->
			<div class="dom-modal-btn-wrap">
				<button type="button" class="btn dom-modal-close-with-click dom-modal-btn-confirm">Ok</button>
				<button type="button" class="btn dom-modal-close-with-click dom-modal-btn-cancel">Cancel</button>
			</div>
			<!-- /for type confirm -->

			<!-- for type info -->
			<div class="dom-modal-btn-wrap">
				<button type="button" class="btn dom-modal-close-with-click dom-modal-btn-info">Ok</button>
			</div>
			<!-- /for type info -->

		</div>
	</div>
</div>

  • modalId - ідентифікатор модального вікна з HTML-коду. З кроку 1.
